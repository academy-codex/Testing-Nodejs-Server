# todoAPI-in-Node.js

## Api implementation for a TodoApp

Link: https://fathomless-gorge-15734.herokuapp.com/ 

## Routes

<b>Signup Route:</b> <br>
Method: Post<br>
Body:<br>
{<br>
"email":"email here"<br>
"password":"password here"<br>
}
<br><br>

<b>Signin Route:</b><br>
Method: Get<br>
Body:<br>
{<br>
"email":"email here"<br>
"password":"password here"<br>
}<br>

Response contains the header <b>'x-auth'</b> which is the authentication access token for the user.
